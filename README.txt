This is a package for tabulating information using basic GUI using '|', '-' and '_' in the python console

syntax:

    import tabulate

    table.create(
        {
        'Name':['John','Peter',],
        'Age' :['12'  ,'26'   ,],
        }   
    )

Result in consloe:
    ________________
   |Name   |Age     |
   |-------|--------|
   |John   |12      |
   |Peter  |26      |
    ────────────────

This needs a lot of development, and this project is hosted on github for developmental purpose, 
in which anyone can contribute, smallest of which are appreciated
